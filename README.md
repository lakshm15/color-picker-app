# Color Picker App

A simple and elegant color picker web application that allows users to select colors, preview them, and save their favorite colors for later use.

## Features

- **Interactive Color Picker**: HTML5 color input for easy color selection
- **Live Preview**: Real-time color display with circular color box
- **Dynamic Background**: Page background updates with selected color gradient
- **Save Colors**: Save your favorite colors to a persistent collection
- **Color Management**: Click saved colors to reselect, delete individual colors, or clear all
- **Local Storage**: All saved colors persist between browser sessions

## Demo

Open `index.html` in your web browser to start using the color picker.

## Files

- `index.html` - Main HTML structure
- `style.css` - Styling and responsive design
- `script.js` - JavaScript functionality and local storage management

## How to Use

1. Use the color picker input to select any color
2. Watch the color box and background update instantly
3. Click "Save Current Color" to add it to your collection
4. Click any saved color swatch to reselect it
5. Hover over saved colors to reveal delete buttons
6. Use "Clear All" to remove all saved colors

## Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- Vanilla JavaScript
- Local Storage API
